# HR Analytics Dashboard (Power BI)

A comprehensive **HR Analytics Dashboard** built with **Power BI** to help organisations monitor workforce insights — headcount, attrition, age distribution, job satisfaction, education-field impact and more. This repository contains documentation, dataset suggestions, and the Power BI report/template used to build the dashboard.

## Project overview
This project contains everything needed to reproduce a modern HR Analytics dashboard in Power BI. The dashboard focuses on core HR KPIs and exploratory visuals for attrition analysis, age and gender breakdowns, job satisfaction by role, and education-field performance.

## Dashboard preview
A preview of the dashboard layout and sample values is included and the uploaded PDF (`HR Analytics Dashboard.pdf`). Replace those sample values with your dataset after loading.

---

## Purpose

The purpose of this repository is to provide a complete, Power BI–based HR Analytics Dashboard solution that enables data-driven decision-making. It consolidates HR data into meaningful visual insights to help HR teams monitor employee trends, identify attrition drivers, evaluate workforce composition, and support strategic planning.
